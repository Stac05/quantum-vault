# QuantumVault

Adaptive Chunk-Based Hybrid Encryption Using ML-KEM-768 and AES-256-GCM for Secure Large File Protection.

## Overview

QuantumVault is an academic research prototype for secure large-file protection using a hybrid encryption approach.
